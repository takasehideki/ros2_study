# ros2_study
ROS 2講習会用のソースファイル

## 目次・内容

項目名はブランチ名に対応しています． 適宜で `git checkout` してご確認ください．

- ROS 2プログラミングの基礎
  - step1: ワークスペースの設定・topic通信の実装
  - step2: 独自定義型のライブラリパッケージの実装
  - step3: 独自定義型を利用したservice通信の実装
- シミュレータによるロボット開発の実践
  - task1: ロボットを操作するパッケージの実装
  - task2: ロボットを操作して前後の位置情報を返すサービスサーバの実装
